1. Open Google

---

## 📊 Dataset Description
This dataset contains information about employees working for the City of Chicago, including:
- Employee names  
- Job titles  
- Departments  
- Salaries  

It provides a comprehensive view of workforce distribution and salary structure across different roles and departments.

---

## 🛠️ Usage in This Project
- The dataset was cleaned using **Excel Power Query**  
- Transformed data was loaded into **SQL** for analysis  
- Insights were generated using SQL queries  
- Final results were visualized using **Excel dashboards**
